GitHub User Finder

Автор: Петченко Виктория 

Как использовать
Введите имя пользователя в поле.
Нажмите «Найти».
Если пользователь существует, вы увидите его имя, количество репозиториев и другую информацию.
Нажмите «⭐ Добавить в избранное», чтобы сохранить пользователя. Список сохраняется в файл favorites.json.
Примеры
Успешный поиск: введите torvalds, нажмите «Найти». Появится информация о Линусе Торвальдсе.
Пользователь не найден: введите несуществующее имя, например asdf1234, и нажмите «Найти». Приложение покажет ошибку.
Пустое поле: попробуйте нажать «Найти», ничего не вводя. Появится предупреждение.
